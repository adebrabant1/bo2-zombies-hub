# Aether French Zombies Hub

Site statique GitHub Pages inspiré de l'ambiance BO2 Zombies, avec code original.

## Installation
1. Crée un dépôt GitHub.
2. Envoie tous les fichiers de ce dossier.
3. Active GitHub Pages : Settings > Pages > Deploy from branch > main > root.

## Assets à placer
- Police : `assets/fonts/bo2.ttf` si tu as le droit de l'utiliser.
- Musique principale : `assets/audio/menu-theme.mp3`
- Son hover : `assets/audio/hover.mp3`
- Images d'atouts : `assets/images/perks/`
- Plans de maps : `assets/images/maps/`
- Images d'armes : `assets/images/weapons/`

## Note droits
Ne pas redistribuer d'assets protégés sans autorisation. Le code du site est original.
